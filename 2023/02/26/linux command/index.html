Linux 基本命令与 Shell 脚本(1)

发表于 2023-02-26 更新于 2023-03-06 分类于 Some Notes 阅读次数：

开个小坑，记一下要用的脚本命令啥的

pwd —打印出当前工作目录名

cd —更改目录

ls —列出目录内容

cd /usr/bin——回到绝对路径下的目录

cd ..——回到上级目录

ls——在当前目 录下所有文件和子目录的列表

ls /usr 查看指定目录下的文件

ls -1 查看目录下文件的更多信息

ls -a 列出所有文件

ls -h 以字节数显示文件大小

ls -t 按照修改时间

ls -r 反向排序

less filename 浏览文件内容

cp —复制文件和目录

mv —移动/重命名文件和目录

mkdir —创建目录

rm —删除文件和目录

ln —创建硬链接和符号链接

type –说明怎样解释一个命令名

which –显示会执行哪个可执行程序

man –显示命令手册页

apropos –显示一系列适合的命令

info –显示命令 info

whatis –显示一个命令的简洁描述

alias –创建命令别名

cat －连接文件

sort －排序文本行

uniq －报道或省略重复行

grep －打印匹配行

wc －打印文件中换行符，字，和字节个数

head －输出文件第一部分

tail - 输出文件最后一部分

tee - 从标准输入读取数据，并同时写到标准输出和文件

clear - 清空屏幕

history - 显示历史记录

grep “cm-1” OUTCAR-查看频率

grep “f/i” OUTCAR 查看虚频

tail -f OSZICAR 查看收敛能量

grep E-fermi OUTCAR 查看费米能级

grep irreducible OUTCAR 查看倒格矢

grep ZVAL OUTCAR 查看价电子

grep TIT POTCAR 检查赝势
