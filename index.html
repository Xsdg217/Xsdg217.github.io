Nobel Lecture: Electronic structure of matter—wave functions and density functionals

W. Kohn
Department of Physics, University of California, Santa Barbara, California 93106

I. INTRODUCTION

​ 在薛定谔的电子波函数方程被提出并在像He和H2这样的小系统上得到了令人惊叹的验证之后不久，狄拉克宣称化学已经走到了尽头——它的内容完全包含在那个强大的方程中。他还补充说，在几乎所有情况下，这个方程都太过复杂，无法求解。

​ 本文的主要目的是解释DFT，这是电子结构理论的另一种方法，其中电子密度分布n(r)，而不是多电子波函数起着中心作用。

​ DFT对多粒子量子系统科学做出了两种贡献，包括分子和凝聚态的电子结构问题:

​ 首先是在基本理解方面：人们已经习惯于用截断的单粒子轨道希尔伯特空间来思考问题。然而，当要求高精度时，需要太多的斯莱特行列式(在一些高达$10^9$的计算)，这使得理解变得困难。DFT 提供了一个互补的视角。它关注真实三维坐标空间中的量，主要是电子密度 $n(r)$。其他令人感兴趣的量是:交换相关空穴密度$n_{xc}(r,r’)$，它描述了在$r$点的一个电子的存在如何耗尽在 $r’$点的其他电子的总密度；以及线性响应函数$\chi\left(r, r^{\prime} ; \omega\right)$，它描述了r点的总密度随频率w而因r’点的摄动势而发生的变化。

​ 第二个贡献是实用性。当传统的多粒子波函数方法应用于多粒子系统时，当原子数N超过一个临界值时，对于一个不对称的系统来说，这个临界值目前在$N_0 =10$附近(到大约2的因子内)，就会遇到我所说的指数壁。按照目前的思路，这些方法在分析或计算方面的重大改进只会导致$N_0$的适度增长。

​ 在第一节中，我将讨论传统的波函数方法，并将它们在处理极少原子系统时的巨大成功与处理非常多原子系统时的基本局限性进行对比；

​ 第二节讨论了波函数方法背景下的离散泛函。首先描述了原始的Hohenberg-Kohn (HK)变分原理，其中n(r)为变分变量。这是紧随其后的K-S自洽的单粒子方程涉及定义良好但在此点未知的交换相关泛函Exc[n(r)]。原则上，当与精确的Exc一起使用时，这些单粒子方程包含了所有多体效应。$n_{xc}(r,r’)$ 依赖于 $\mu - V_{eff}(r’)$,$r->r’$.接下来简要讨论了Exc的近似值；

​ 第三-六节讨论了DFT在电子基态上的应用，以及对其他电子和非电子系统的大量推广；

​ 最后，提出了一些结束语和猜想。

III. DENSITY-FUNCTIONAL THEORY—BACKGROUND

​ 起源是 托马斯费米(TF)理论，它考虑了在外势v (r)中运动的相互作用电子，并提供了v (r)和密度分布n(r)之间的(高度简化的)一对一隐式关系:

​ 在随后的几年里，引入了各种改进方法(梯度修正、交换修正和相关修正)，但该理论在应用于物质的电子结构方面并没有明显变得更有用。

​ 提出问题: 从原则上讲，用n(r)来完整、准确地描述电子结构是可能的吗?一个关键问题是密度n(r)是否完全表征了系统。于是提出了一个假设，即任何电子系统(有或没有相互作用)的基态密度n(r)的知识唯一地决定了该系统。这一假设成为现代DFT的起点。

IV. THE HOHENBERG-KOHN FORMULATION OF DENSITY-FUNCTIONAL THEORY

A. The density n(r) as the basic variable

​ The basic lemma of HK.：在某个外势v (r)中相互作用电子的束缚系统的基态密度n(r)唯一地决定了这个势(Hohenberg, Kohn, 1964).

​ 注:(1)术语“唯一”在这里的意思是:不超过一个unintersting的附加常数. (2)在简并基态的情况下，引理指任意基态密度n(r). (3)这个引理在数学上是严谨的。

VI. GENERALIZATIONS AND QUANTITATIVE APPLICATIONS

A. Generalizations

​ 自旋极化系统的自旋DFT；简并基态；多组分体系(电子空穴液滴、原子核)；M简并基态的集合DFT；有限温度下的自由能；具有电子配对机制的超导体；相对论性电子；激发态；电流密度泛函理论抗磁性；玻色子(而非费米子)；含时演化；DFT与分子动力学或蒙特卡罗方法的结合

B.Application

​ 一个例子：碱金属的自旋敏感性. LSDA. 精确 1%

​ 梯度修正和/或混合方案的加入，使大规模化学应用的计算能量通常提高了几乎一个数量级;在物理应用中，这种改进通常没有那么显著。几何参数的精度保持在1%水平。

VII. CONCLUDING REMARKS

​ DFT已经被物理学家和化学家广泛接受。对于周期性固体，有时称为标准模型。在化学中，DFT补充了传统的基于波函数的方法，特别是对于原子数量非常多的系统。

​ 在DFT目前工作仍然相当差的情况下(例如，远距离极化能量;消失电子密度区域;部分填充的电子壳;反应障碍)，离散傅里叶变换常常为我们目前对真实空间坐标中的电子结构的理解需要如何改进提供线索。

​ 展望未来，希望基于波函数的理论和基于密度的理论，将以互补的方式，继续不仅给我们定量上更精确的结果，而且还有助于更好地理解物质的电子结构的物理/化学。

  • 简要思路分析：

    对 github 上的 finite system algorithm 部分进行改写。其中主要思路如下：

    1. 根据哈密顿量形式定义单体和两体的 H，定义 block 数据类型作为传递；

    2. enlarge_block 函数实现向当前的 block 添加一个节点的“扩展”，主要考虑三部分变化：current block 内的两体哈密顿量， 新节点内的单体哈密顿量，以及二者连接处的两体哈密顿量；

    3. 单步DMRG，执行一个DMRG步骤，使用’ sys ‘作为系统，’ env ‘作为环境，截取 m 个基矢；

    4. Traditional finite DMRG algorithm， 先做 infinte 的 DMRG 的 enlarge， 将结果存到 block_disk 中方便后面调用。然后做 full size 的 finite DMRG，即 sweep back and forth 直到收敛

  • 对源代码的修改主要包括几个部分：

    1. 哈密顿量单体和两体作用，以及后面调用接口的修改；

    2. 密度矩阵 rho 作为结果 return， 并计算纠缠熵。我们知道矩阵的迹等于其特征值的和，据此来计算 S(L)；

    3. 收敛判据。原代码对于每一个 m ，做了一个周期的 sweep，这里将收敛判据改成 E/L 的收敛情况（尽管经过测试，原代码也收敛）；

    4. 其他（例如L,m_warmup参数修改等 trivial 的地方）。

  • 第一问：m =30 ，g 取 [0.7,1.0,1.7]

  • 数组 EE 储存的是 [s, sys_block.length, m, g ], 分别对应：纠缠熵， 系统block长度，截断m，参数g。

1. 问题描述

​ 在用markdown写文档时，若包含数学公式，渲染会出现一些问题。常用的Markdown编辑器都会集成Mathjax，用来渲染文档中的类Latex格式书写的数学公式，基于Hexo搭建的个人博客，需要对渲染文档进行一些修改。

2. 原因

​ Hexo 默认使用 hexo-renderer-marked 引擎渲染网页，该引擎会把一些特殊的 markdown 符号转换为相应的 html 标签，比如在 markdown 语法中，下划线_代表斜体，会被渲染引擎处理为 $em$ 标签。

​ 因为类 Latex 格式书写的数学公式下划线_表示下标，有特殊的含义，如果被强制转换为 $em$ 标签，那么 MathJax 引擎在渲染数学公式的时候就会出错。类似的语义冲突的符号还包括*, {, }, \\等。

Homework (task 1)
  • write a code for diagonalizing the extended Hubbard model on a 6-site ladder
$$ H=-t \sum_{\{i, j\}, \sigma}\left(C_{i, \sigma}^{\dagger} C_{j, \sigma}+C_{j, \sigma}^{\dagger} C_{i, \sigma}\right)+U \sum_i n_{i \uparrow} n_{i \downarrow}-V \sum_{\{\{i, j\}\}}\left(n_{i, \uparrow}+n_{i, \downarrow}\right)\left(n_{j, \uparrow}+n_{j, \downarrow}\right)-\mu \sum_{i, \sigma} n_{i, \sigma} $$ $$ \begin{aligned} &\{i, j\} \in(0,1),(1,2),(3,4),(4,5),(1,3),(2,4) \\ &\{\{i, j\}\} \in(0,3),(1,4),(2,5),(2,3) \end{aligned} $$
  • take t = 1.0, U = 1.4, V = 0.3, 𝜇 = 0.1

(1) show the lowest 20 eigenvalues. please make use of the U(1) symmetry

(2) calculate $\left\langle n_{i, \sigma}\right\rangle$ for each site and spin using the ground state

  • 简单思路分析：

    1. 六个site，考虑spin的up和down，则可以用一个12位的二进制数表示一个态。规则如下：

      | xxxxxx xxxxxx > : 左边表示自旋朝下，右边表示自旋朝上，每部分从右往左分别编号为 0，1…,5

      例如 ： | 000010 000001 > 表示在 0 site 有一个$\uparrow$，在 1 site 有一个$\downarrow$, 其他位置为空

    2. 哈密顿量分为对角项（u，v，𝜇）和非对角项（t）：
